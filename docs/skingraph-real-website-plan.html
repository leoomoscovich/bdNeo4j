Control Contract

These are the expected behaviors for every currently decorative interaction.

Sidebar

  • Workspace buttons switch the visible product area.
  • Marketplace toggles filter dashboard, radar and cycles.
  • Run deep scan calls POST /api/scans.

Topbar

  • Search updates global query filters.
  • Filter pills reflect real active filters.
  • Clearing search restores unfiltered results.

Hero Actions

  • Open best opportunity selects the top ranked deal.
  • Inspect risk cycles switches to the risk workspace.
  • Compare markets opens the comparison workspace.

Drawer Actions

  • Track toggles local watchlist.
  • Open graph switches to graph explorer.
  • Compare adds item to comparison.
  • Export downloads JSON or CSV.

API Expansion

The website needs backend behavior for scans, traders, comparison and export.

GET /api/tradersTrader ranking and search
GET /api/traders/[id]Trader profile and graph context
POST /api/scansCreate a deep scan and persist summary
GET /api/scans/latestRead latest scan status
GET /api/compare?ids=...Compare selected assets or opportunities
GET /api/exportDownload JSON or CSV from filtered data
